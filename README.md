# 공사비닷컴 복원형 초안

기존 운영 사이트의 공개 검색 결과와 현재 확인 가능한 기능명을 기준으로, 업로드된 PHP 초안을 공사비닷컴 원형 구조에 가깝게 재구성한 버전입니다.

## 반영 범위

- 상단 유틸 메뉴, 로고, 6개 주 메뉴, 드롭다운 구조
- 회사소개 / 보고서 / 공사비 자료 / 공사비 교육 / 커뮤니티 / 고객센터 URL 구조
- 면적당 공사비 검색, 공사 단가 검색, 골조 면적별 수량 검색 화면
- 비회원/무료회원 샘플 검색 및 유료회원 전체 검색 안내 문구
- 보고서 메뉴의 `서비스 오픈 준비중 입니다.` 화면
- 회원가입의 무료회원/유료회원 구분 및 연 1,000,000원(VAT 별도) 안내
- 공사비 교육 / 교재 안내 / 동영상 강의 화면
- 건설 장터 / 공사비 작성 의뢰 / Partners / 구인·구직 화면
- 고객센터 게시판형 화면
- 푸터 사업자 정보

## 한계

기존 서버의 PHP 원본, DB, 첨부파일, 배너 원본 이미지는 포함되어 있지 않습니다. 100% 픽셀 단위 복구를 위해서는 호스팅 업체로부터 다음 자료가 필요합니다.

1. 웹 루트 전체 백업
2. DB 전체 `.sql` 백업
3. 업로드 폴더 전체
4. 관리자 계정 또는 관리자 페이지 백업
5. 기존 이미지/배너/로고 원본

## 실행

```bash
cp .env.example .env
php database/migrate.php
php -S 127.0.0.1:8000 -t public
```

브라우저에서 `http://127.0.0.1:8000` 접속.
